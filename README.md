# 🌿 Vermutier — MaestroBot

Herramienta de formulación artesanal de Gin y Vermouth.

**Funcionalidades:**
- Formulación de botánicos en g/L con roles (backbone, cuerpo, acabado)
- Escala automática de batch con cálculo de mermas por estilo
- Análisis de costos y precios de venta sugeridos por canal
- Ficha de cata con radar chart y puntuación global
- Recetario persistente en localStorage

**Stack:** React + Vite · Deploy en GitHub Pages

**URL:** https://francisco-vila.github.io/vermutier/
